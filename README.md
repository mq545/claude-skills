# Claude Skills

Production-ready skills for Claude that solve real problems.

## Skills

### Skill Finder
Discovers which skills you should create by analyzing your conversation history for repeatable patterns.

**What it does**: Searches your conversations, identifies tasks you do repeatedly (3+ times), and recommends which ones would benefit from being formalized into skills.

**Use it for**: Understanding your actual Claude usage patterns, prioritizing which skills to build, finding workflow optimization opportunities.

## Installation

1. Clone this repo or download the skill folder you want
2. Open Claude.ai and go to your project
3. Drag and drop the skill folder into the chat
4. Ask: "What skills should I create?"

---

## Requirements

- Claude Pro or Claude for Work
- Project with conversation history (20+ conversations recommended)
